Sku - Screen Reader Interface for World of Warcraft

Sku is a World of Warcraft addon that provides a complete screen reader interface, making the game accessible for blind and visually impaired players. It works with screen readers like NVDA and JAWS.

Notice: The addon versions on this page are for World of Warcraft 2.x – The Burning Crusade Anniversary Server. A page for the Classic Era version will follow soon.

Required Downloads

You need both of these addons to use Sku. Download and install them into your WoW AddOns folder.

Sku (Main Addon) - Version 41.02.05

The core addon that provides the screen reader interface.

Download Sku v41.02.05

SkuBeaconSoundsets

Required sound sets for Sku's audio beacon navigation system.

Download SkuBeaconSoundsets

Language Pack (choose one)

Choose the language pack that matches your preferred language. You only need one of these.

English

Download SkuAudioData English

German (Deutsch)

Download SkuAudioData German

German Fast (Deutsch schnell)

German language pack with faster speech output.

Download SkuAudioData German Fast

Optional Extensions

These are additional sound packs that add more beacon sounds. They are not required to use Sku.

SkuCustomBeaconsEssential

A small collection of essential custom beacon sounds.

Download SkuCustomBeaconsEssential

SkuCustomBeaconsAdditional

A larger collection of additional custom beacon sounds.

Download SkuCustomBeaconsAdditional
